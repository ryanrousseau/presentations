What is Continuous Integration?

Merging code often to a shared repository

Automated builds and tests trigger on commit

What about Continuous Delivery and Continuous Deployment?

Continuous Delivery means that the software deployment process is automated

Every change can be deployed to production

Continuous Deployment is an extension of Continuous Delivery

Every change is automatically deployed to production

Continuous Deployment requires a lot of infrastructure, tooling, and discipline

Continuous Deployment's not for everyone

Simple CI/CD Process

  • Package the applications
  • Deploy the applications
